# VTaper

A complete Android app built with Kotlin and Jetpack Compose.
Featuring two core pillars: V-Taper Physique Builder and Dopamine Reset (Reboot Mode).

## Setup
1. Open Android Studio.
2. Select "Open" and choose the `VTaper` directory.
3. Allow Android Studio to sync and download dependencies.
4. Run on an emulator or physical device.

## Features
- **Design System**: Onyx & Violet theme (Dark mode only).
- **Onboarding**: 8-page guided setup.
- **Workout**: Exercise cards with form cues and RPE.
- **Nutrition**: Meal plans and macro tracking.
- **Reboot Mode**: Streak tracker and urge surfing tool (Protected with FLAG_SECURE).
